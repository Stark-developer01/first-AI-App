# first-AI-App

A static web dashboard for tracking and comparing global technology hiring trends with the Indian job market.

## Features
- Home dashboard with role-wise, experience-wise, salary range, and tech stack demand comparison.
- Separate page focused on global hiring and job market pattern signals.
- Visual representation using charts, graphs, KPI cards, and a comparison table.
- Clean responsive interface for desktop and mobile.

## Run locally
From the repository root:

```bash
python3 -m http.server 8080
```

Open:
- `http://localhost:8080/index.html`
- `http://localhost:8080/global-patterns.html`
